# 🍳 SmartRecipe — Cook Smart. Eat Smart.

SmartRecipe is a modern, interactive web application that helps users **discover recipes, explore cuisines, track calories, and follow personalized diet plans** — all in one beautifully designed interface.

---

## 🚀 Features

### ✨ 1. Splash Screen Experience

* Animated intro with video background
* Typing text animation
* Smooth entry transition into the app

---

### 🍽️ 2. Recipe Search System

* Search recipes by:

  * Dish name
  * Ingredients
  * Cuisine
* Fetches real-time data using **TheMealDB API**
* Displays multiple recipe results dynamically

---

### 🌍 3. Explore Cuisines

* Browse recipes by cuisine:

  * Indian 🇮🇳
  * Italian 🇮🇹
  * Mexican 🇲🇽
  * Japanese 🇯🇵
  * Chinese 🇨🇳
  * Thai 🇹🇭
  * Mediterranean 🌿
  * American 🇺🇸

---

### 🔥 4. Trending Recipes

* Displays popular recipes dynamically
* Loading animation while fetching data

---

### ❤️ 5. Favorites System

* Save your favorite recipes
* Stored locally for quick access
* View saved recipes anytime

---

### 🥗 6. Diet Plan Generator

* Choose your goal:

  * 🔥 Weight Loss (~1500 kcal/day)
  * 💪 Muscle Gain (~2500 kcal/day)
  * ⚖️ Balanced (~2000 kcal/day)
* Automatically suggests meals based on your goal

---

### 📊 7. Calorie Tracker

* Set daily calorie goal
* Track:

  * Total calories consumed
  * Progress percentage
* Circular progress indicator
* Add calories manually
* Daily log of intake

---

### 🧬 8. Macro Breakdown

* Tracks:

  * Protein
  * Carbohydrates
  * Fat
* Visual progress bars for each macro

---

### 💡 9. Smart Meal Suggestions

* Suggests meals based on remaining calories

---

### 🎨 10. Modern UI/UX

* Smooth animations (GSAP-based)
* Responsive design
* Dark theme support
* Floating ingredient background
* Interactive components

---

## 🛠️ Tech Stack

* **HTML5** — Structure
* **CSS3** — Styling & animations
* **JavaScript (Vanilla JS)** — Logic & interactivity
* **GSAP (GreenSock)** — Advanced animations
* **TheMealDB API** — Recipe data
